In Framer Motion, le transizioni e animazioni possono essere gestite utilizzando un'ampia varietà di curve di easing. Le curve di easing determinano la velocità e il comportamento di una transizione, facendo sì che l'animazione sia più fluida e naturale.

Ecco una panoramica dei principali tipi di easing che puoi usare in Framer Motion, molti dei quali derivano dalle curve di CSS easing:

1. Linear

  • L'animazione si muove alla stessa velocità per tutta la durata.

javascript
ease: "linear"

2. Ease In

  • L'animazione parte lentamente e poi accelera nel corso del tempo.

javascript
ease: "easeIn"

3. Ease Out

  • L'animazione inizia velocemente e poi rallenta alla fine.

javascript
ease: "easeOut"

4. Ease In Out

  • L'animazione inizia lentamente, accelera e poi rallenta di nuovo prima di fermarsi.

javascript
ease: "easeInOut"

5. Custom Cubic Bezier

  • È possibile definire curve di easing personalizzate utilizzando i valori di una curva di Bezier cubica (4 valori numerici).

javascript
ease: [0.42, 0, 0.58, 1]  // equivalente a "easeInOut" in CSS

6. Back

  • Il movimento oltrepassa leggermente il suo valore finale e poi torna indietro, creando un effetto di "ritorno".

javascript
ease: "backIn" // o "backOut"

7. Bounce

  • L'animazione si muove come se rimbalzasse, dando l'effetto di un oggetto che colpisce una superficie e rimbalza.

javascript
ease: "bounceIn" // o "bounceOut"

8. Elastic

  • L'animazione segue un movimento elastico, come una molla.

javascript
ease: "elasticIn" // o "elasticOut"

9. Spring

  • Una fisica più naturale, con un effetto che sembra imitare una molla.

javascript
ease: "spring" // La velocità, la rigidità e l'ammortizzazione possono essere personalizzate

10. Circ

  • Un effetto di easing che simula un movimento che segue la curva di un cerchio.

javascript
ease: "circIn" // o "circOut"

11. Expo

  • Utilizza una curva esponenziale per un rapido inizio e una fine più graduale.

javascript
ease: "expoIn" // o "expoOut"

Esempio di codice:

javascript
import { motion } from 'framer-motion';

const MyComponent = () => (
  <motion.div
    initial={{ opacity: 0 }}
    animate={{ opacity: 1 }}
    transition={{ duration: 1, ease: "easeInOut" }}
  >
    Ciao, Framer Motion!
  </motion.div>
);

Ogni tipo di easing ha il suo scopo, a seconda di come desideri che le animazioni si comportino durante la transizione. Puoi scegliere quello che meglio si adatta al comportamento visivo che vuoi ottenere.

  1. Usare flexbox o grid per disporre gli elementi orizzontalmente.

  2. Abilitare lo scroll orizzontale usando overflow-x: auto in CSS.

  3. Aggiungere animazioni di scorrimento per rendere l'effetto visivamente interessante.

    "Title 10"

Scroll automatico (opzionale):

Se vuoi che i titoli si spostino automaticamente da destra a sinistra (ad esempio per un effetto di scorrimento continuo), puoi usare un'animazione con Framer Motion o CSS. Ecco come fare con Framer Motion:

javascript
import React from "react";
import { motion } from "framer-motion"; // Per animazioni con framer-motion

const ScrollableTitles = () => {
  const titles = [
    "Title 1",
    "Title 2",
    "Title 3",
    "Title 4",
    "Title 5",
    "Title 6",
    "Title 7",
    "Title 8",
    "Title 9",
    "Title 10"
  ];

  return (
    <div className="scroll-container">
      <motion.div
        className="scroll-titles"
        animate={{ x: "-100%" }}
        transition={{
          x: {
            repeat: Infinity,
            repeatType: "loop",
            duration: 20, // Durata del ciclo di scroll
            ease: "linear"
          }
        }}
      >
        {titles.map((title, index) => (
          <motion.div
            key={index}
            className="title"
            whileHover={{ scale: 1.1 }} // Scaling dei titoli al passaggio del mouse
            transition={{ duration: 0.2 }}
          >
            {title}
          </motion.div>
        ))}
      </motion.div>
    </div>
  );
};

export default ScrollableTitles;

In questo caso, l'animazione si sposterà automaticamente da destra a sinistra con un effetto continuo. Puoi personalizzare la durata e la velocità del ciclo cambiando il valore di duration.

import { motion } from "framer-motion"; // Per le animazioni sui link

const ScrollableLinks = () => {
  const links = [
    { label: "Home", to: "/" },
    { label: "About", to: "/about" },
    { label: "Services", to: "/services" },
    { label: "Contact", to: "/contact" },

    { label: "FAQ", to: "/faq" },
    { label: "Support", to: "/support" },
    { label: "Testimonials", to: "/testimonials" }
  ];

  // Stato per tenere traccia della posizione corrente nel carosello
  const [scrollPosition, setScrollPosition] = useState(0);

  // Funzione per muovere il carosello a sinistra
  const scrollLeft = () => {
    setScrollPosition((prev) => prev - 200); // Spostamento a sinistra (200px per esempio)
  };

  // Funzione per muovere il carosello a destra

      <button onClick={scrollLeft} className="scroll-button left">

      </button>

        className="scroll-links"
        style={{ transform: `translateX(${scrollPosition}px)` }} // Muove i link

            transition={{ duration: 0.3 }}
          >
            <Link to={link.to} className="link-text">
              {link.label}
            </Link>
          </motion.div>

      </motion.div>

      </button>

  return <div>Contenuto del Componente 3</div>;

Spiegazione

  1. Funzione di aggiornamento dello stato: La funzione che passi a setIndice deve restituire il nuovo valore dello stato. Non puoi usare un if senza un return che restituisce un valore.

  2. Controllo del valore: Stai cercando di fare un controllo sul valore di prevIndice. Se prevIndice è 0, restituisci l'ultimo indice dell'array (componenti.length - 1). Altrimenti, decrementi prevIndice di 1 per ottenere l'indice precedente.

  return <div>Contenuto del Componente 1</div>;
}

  return <div>Contenuto del Componente 2</div>;
}

function Componente3() {
  return <div>Contenuto del Componente 3</div>;
}

function Carosello() {
  // Array dei componenti
  const componenti = [<Componente1 />, <Componente2 />, <Componente3 />];
  
  // Stato per l'indice del componente corrente
  const [indice, setIndice] = useState(0);

  // Funzione per andare al componente precedente
  const precedente = () => {
    setIndice((prevIndice) => {
      if (prevIndice === 0) {
        return componenti.length - 1; // Se siamo al primo elemento, ritorniamo l'ultimo
      } else {
        return prevIndice - 1; // Altrimenti, decrementiamo l'indice
      }
    });
  };

  // Funzione per andare al componente successivo
  const successivo = () => {
    setIndice((prevIndice) => {
      if (prevIndice === componenti.length - 1) {
        return 0; // Se siamo all'ultimo elemento, ritorniamo al primo
      } else {
        return prevIndice + 1; // Altrimenti, incrementiamo l'indice
      }
    });
  };

  return (
    <div>
      <button onClick={precedente}>Precedente</button>
      
      {/* Mostra il componente corrispondente all'indice */}
      <div>{componenti[indice]}</div>
      
      <button onClick={successivo}>Successivo</button>
    </div>
  );
}
